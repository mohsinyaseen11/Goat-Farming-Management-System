# 🐐 Goat Farming Management System

A simple **Python + Pandas based CRUD project** to manage goat farm records using a CSV file.

This project is designed for learning **data handling, file management, and basic software design concepts** in Python.

---

## 🚀 Features

- ➕ Add new goat record
- 👀 View all goats
- ✏️ Update goat information
- ❌ Delete goat by ID
- 🔍 Search goat by ID
- 💾 Data saved in CSV file

---

## 🛠️ Technologies Used

- Python 🐍
- Pandas 📊
- CSV File Handling

---

## 📂 Project Structure

goat-farming-system/
│── main.py

│── README.md

---

## 📊 Data Fields

Each goat record contains:

- ID  
- Name  
- Breed  
- Gender  
- DOB / Purchase Date  
- Purchase Price  
- Health Status  

---

## ▶️ How to Run

### 1. Install dependencies
pip install pandas

### 2. Run the program
python main.py

---

## 🧠 Project Working

- Data is stored in a CSV file (`goats.csv`)
- Pandas handles data operations
- Menu-based system is used for interaction
- Each function performs a specific task (Add, Update, Delete, Search)
